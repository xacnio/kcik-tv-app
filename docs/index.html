KcikTV

Modern and fast Android TV client for Kick.com

Features

  • Fast Performance: Fluid interface optimized specifically for Android TV hardware.
  • Live Chat Support: Real-time interaction during streams with full emote support.
  • Easy Login: Quick login via QR Code or manual entry with 2FA (OTP) support.
  • Remote Control Optimization: Fully compatible with standard D-Pad TV remotes for intuitive navigation.
  • Ultra Low Latency: Powered by IVS technology for the fastest streaming experience.
